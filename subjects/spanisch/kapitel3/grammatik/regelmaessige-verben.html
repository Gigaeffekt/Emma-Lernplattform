📖 Regelmäßige Verben – Die Logik!

💡 Das Prinzip – ganz einfach!

Spanische Verben enden im Infinitiv auf -AR, -ER oder -IR.
Um sie zu konjugieren: Endung abschneiden → neue Endung dranhängen!

  • cenar → cen- + o/as/a/amos/áis/an
  • comer → com- + o/es/e/emos/éis/en
  • vivir → viv- + o/es/e/imos/ís/en
cenar → cen- + o / as / a / amos / áis / an

cenar – zu Abend essen

escuchar (zuhören) → escuch- + o/as/a/amos/áis/an

comer → com- + o / es / e / emos / éis / en

Weitere -ER Verben

beber (trinken) → beb- + o/es/e/emos/éis/en
leer (lesen) → le- + o/es/e/emos/éis/en
Achtung bei leer: leo (nicht "leo" klingt aber wie "ley-o")

vivir → viv- + o / es / e / imos / ís / en

vivir – wohnen / leben
