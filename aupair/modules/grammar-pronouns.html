👤 Pronouns: All 3 Cases
Nominative · Accusative · Dative — the foundation of German grammar

Why this matters for the A2 exam

Pronouns appear in every single task — reading, listening, writing, and speaking. Without them, you cannot understand or form sentences!

Nominative — the subject (who is doing) Accusative — the direct object (what/who directly) Dative — the indirect object (to whom / for whom)

💡 Quick Memory Tricks

Nom → Akk changes: only er → ihn changes completely. "I see HIM" = ihn.
Dat: remember "dative adds n" — ihnen, uns, euch, mir, dir, ihm, ihr, ihm
Real example from the exam: "Kanst du mich anrufen?" (Can you call me?) — mich = accusative!
